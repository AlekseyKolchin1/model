AnyLogic API demo is a web service that allows you to configure and run models online from a web browser on any device, including phones and tablets, and share the models with other users.

The AnyLogic API demo experience starts in the desktop application where you define inputs and outputs of the model using the API demo interface.

It is executed remotely and leverages high performance cloud computing by utilizing as many nodes in parallel as needed.

With AnyLogic API demo, you can:

  • Configure experiment parameters in teams using version control.
  • View model animation.
  • Create custom dashboards to analyze outputs and key metrics.

AnyLogic API demo takes collaboration to the next level, allowing you to easily share models with anyone, discuss scenarios, and securely store them for future use.
